# Canvas Design Skill

Create sophisticated visual art through a two-phase process: developing a design philosophy, then expressing it as museum-quality `.pdf` or `.png` artifacts.

## Core Process

### Phase 1 — Philosophy Creation
Develop a named aesthetic movement (1-2 words) articulated through 4-6 paragraphs describing how ideas manifest visually via:
- Space and negative space
- Form and geometry
- Color relationships
- Composition and balance
- Visual hierarchy

Text remains minimal and integrated as visual accent, not explanation.

### Phase 2 — Canvas Expression
Transform the philosophy into a single-page (or multi-page) composition emphasizing visual communication over explanation. Typography serves design-forward purposes with proper spacing and containment.

## Key Principles

- Final work must appear "meticulously crafted" — the product of deep expertise
- Design operates through spatial relationships and visual hierarchy
- Every detail must be intentional and perfect
- Work should be suitable for museum, magazine, or professional presentation contexts

## Critical Requirements

- Original designs only (no copyright violations)
- Text as sparse, essential elements
- Flawless formatting with proper margins
- No overlapping elements
- Sophisticated aesthetic regardless of subject matter
- Output as downloadable PDF or PNG

## Output Formats

- **PNG** — For web use, social media, presentations
- **PDF** — For print, multi-page layouts, professional delivery
